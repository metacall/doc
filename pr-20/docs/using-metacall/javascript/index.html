JavaScript

Works for JavaScript, TypeScript, NodeJS.

Once you have installed the MetaCall CLI globally using the installation guide, you can start integrating MetaCall into your JavaScript/Node.js projects. Follow the steps below to set up MetaCall in your Node.js environment.

NPM Package​

  • npm :

    metacall npm install metacall

Importing MetaCall​

To import MetaCall in your Node.js project, use the following :

const {
    metacall,
    metacall_load_from_file
    metacall_inspect,
    metacall_load_from_file_export,
    metacall_load_from_memory,
    metacall_load_from_memory_export,
    metacall_handle,
    metacall_logs
} = require('metacall');

Function Definitions​

  1. metacall(name, ...args)​

    This function executes a function which you have imported from another file (which can be a Python function, C++ function etc)

    Parameters

    • name (String) : Name of the function to call.

    • args (any) : Arguments to pass to the function.

    Returns : The result of the function execution.

    Example :

    • index.js
    • sum.py
    const { metacall, metacall_load_from_file } = require('metacall');

    metacall_load_from_file('py', ['sum.py']);

    console.log(metacall('add', 5, 10));
    def add(a, b):
        return a + b

  2. metacall_load_from_file(tag, paths)​

    This function loads a file / script into your JS file. (imported file can be a Python file, C++ file etc)

    Parameters

    • tag (String) : Loader Type (e.g 'py' for Python, 'js' for JavaScript).

    • paths (Array) : File paths to be loaded.

    Returns : Load Result.

    Example :

    metacall_load_from_file('py', ['sum.py']);
